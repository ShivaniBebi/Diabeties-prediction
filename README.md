# 🩺 Diabetes Prediction System

An AI-powered **Diabetes Prediction System** built using **Machine Learning** and **Streamlit**. The application predicts whether a patient is at risk of diabetes based on medical parameters and provides an intuitive dashboard with probability scores, risk analysis, and personalized health recommendations.

---

## 📌 Features

* 🤖 Machine Learning-based diabetes prediction
* 📊 Interactive Streamlit web application
* 📈 Risk probability gauge chart using Plotly
* 🔍 Patient risk factor analysis
* 💡 Personalized health recommendations
* 📱 Responsive and user-friendly interface
* ⚡ Real-time predictions

## 📊 Dataset

The project uses the **Pima Indians Diabetes Dataset**, containing **768 patient records** with the following features:

* Pregnancies
* Glucose
* Blood Pressure
* Skin Thickness
* Insulin
* BMI
* Diabetes Pedigree Function
* Age

Target Variable:

* **0 → Non-Diabetic**
* **1 → Diabetic**

---

## ⚙️ Machine Learning Workflow

1. Load the dataset
2. Perform data preprocessing
3. Feature scaling using **StandardScaler**
4. Split data into training and testing sets
5. Train the models
6. Evaluate performance
7. Save the trained model
8. Deploy using Streamlit
